File that will be added to IPFS

Variants§

§

Path(PathBuf)

Path in local disk storage to the file.

§

Stream((Option<String>, Vec<u8>))

Stream of file bytes, with an optional name. NOTE current implementation of rust-ipfs does not add names to published files.
