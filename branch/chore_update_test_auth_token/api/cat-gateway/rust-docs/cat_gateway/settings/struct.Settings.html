pub(crate) fn api_host_names() -> Vec<String>

Get a list of all host names to serve the API on.

Used by the OpenAPI Documentation to point to the correct backend. Take a list of [scheme://] + host names from the env var and turns it into a lits of strings.

Host names are taken from the API_HOST_NAMES environment variable. If that is not set, returns an empty list.
